This program converts large numbers from one number system to another and adds them up. 

Usage:
1) Enter the numeral system of the input data 
2) Enter a system for reading the output 
3) Enter the first term in the input numeral system
4) Enter the second term in the input number system
5) The program will display the result of addition in the output numeral system 

P.S. The program is written with control over the input of all errors by the user, which is why there are so many lines of code.
